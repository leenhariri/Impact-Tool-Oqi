# Contributing 
Contributions are welcome and appreciated, especially those that improve functionality, usability, documentation, or security.

This project is developed within the Open Quantum Institute ecosystem and follows internal quality and review standards.

## How to contribute

-Fork the repository.

-Create a feature branch from the main branch.

git checkout -b feature/your-feature-name

-Make and commit your changes with clear, descriptive commit messages.

git commit -m "Describe your change"

-Push the branch to your fork.

git push origin feature/your-feature-name

-Open a pull request describing the purpose and scope of your changes.

## Contribution Guidelines

-Ensure code follows the existing project structure and style.

-All new features should include appropriate validation and error handling.

-Avoid introducing breaking changes without prior discussion.

-Security-related changes should be clearly documented in the pull request.
